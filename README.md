# cardsTD — トランプ王国タワーディフェンス

トランプ（cards）をモチーフにした、1人用のリアルタイム・タワーディフェンスです。
攻めてくる「カード兵」を、あなたの「トランプ兵士のタワー」で食い止めましょう。
タワーも敵も、カードの体に細い手足と頭を付けた兵士としてキャラクター化しています。

外部ライブラリ・外部素材ゼロの **HTML5 Canvas + バニラJavaScript**。オフラインで動き、
Androidのブラウザでそのまま遊べます（後述のとおりアプリ化も可能）。

## 遊び方

1. `index.html` をブラウザで開く（PCでもスマホでもOK）。
2. 「ゲーム開始」をタップ。
3. 下のカード（タワー）をタップ → 盤面の空きマスをタップで **配置**。
   - ♠ スペード剣兵：高威力・単発
   - ♦ ダイヤ弓兵：連射・長射程
   - ♣ クラブ砲兵：範囲攻撃
4. 「ウェーブ開始」で敵が出現。敵を倒すと **ゴールド** を獲得。
5. 配置済みタワーをタップ → **強化**（J → Q → K に昇格）または **売却**。
6. 敵はゴールに着くと **ライフ** を1削る。ライフ0でゲームオーバー。
7. 全10ウェーブを耐え抜けばクリア。終盤は絵札（J/Q/K）や最強の **エース(A)** の王が襲来します。

※ トランプのランクは一般的な **エース・ハイ**（2 < 3 < … < 10 < J < Q < K < A、A が最強）に準拠。

- 画面上部で **速度切替（x1 / x2）** と **一時停止** ができます。
- マウス・タッチ両対応。

## ファイル構成

```
index.html        画面骨格とスクリプト読み込み
css/style.css     レイアウト・テーマ（カードテーブル調）
js/config.js      マップ・タワー・敵・ウェーブの定義（バランス調整はここ）
js/render.js      カード兵士の描画（手足・頭付き）、盤面・弾・エフェクト
js/game.js        ゲーム状態・エンティティ・シミュレーション
js/ui.js          HUD・カードトレイ・強化パネル・オーバーレイ
js/input.js       マウス/タッチ入力（Pointer Events）
js/main.js        起動・Canvasサイズ調整・ゲームループ
```

ES modules は使わず、複数の `<script>` を `window.CTD` 名前空間で共有しています。
そのため `index.html` を直接開くだけ（`file://`）でも動作します。

## ローカルで動かす

ダブルクリックで `index.html` を開くだけで動きます。簡易サーバーで確認する場合:

```
npx http-server .      # もしくは  python3 -m http.server
```

## Androidアプリ（APK）にする（任意）

Web版が完成しているので、[Capacitor](https://capacitorjs.com/) でネイティブアプリ化できます（ビルドはローカル/クラウドで）:

```
npm init -y
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init cardsTD com.example.cardstd --web-dir=.
npx cap add android
npx cap sync
npx cap open android   # Android Studio で APK をビルド
```
